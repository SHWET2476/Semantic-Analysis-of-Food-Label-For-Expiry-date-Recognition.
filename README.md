# Semantic-Analysis-of-Food-Label-For-Expiry-date-Recognition.
Semantic Analysis of Food Labels for Expiry Date Recognition helps reduce food wastage by automatically detecting and notifying users about approaching expiration dates. 
Using OCR and deep learning techniques, it extracts expiry details from food labels and triggers timely alerts through mobile and web interfaces. 
The solution includes a food inventory dashboard for managing and tracking expiration dates efficiently. This project aims to promote sustainability and minimize waste by ensuring proactive consumption habits.
